## Method Signature

```typescript
createCheckout(checkoutRequestObject: CreateCheckoutObject): Promise<void>
```

This method redirects the user to PayMaya Checkout, where they can finalize their payment. The method accepts a checkout request object and returns a promise that resolves when the redirect is initiated.
