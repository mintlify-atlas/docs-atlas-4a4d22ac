## Method Signature

```typescript
addTransactionHandler(callback: (paymentTokenId: string) => void): void
```

## Overview

The `addTransactionHandler()` method registers a callback function that will be invoked when a user successfully submits their credit card information through the form created by `createCreditCardForm()`. The callback receives a payment token ID that can be used to process the payment on your server.

This method sets up a message listener that securely receives tokenized payment data from the PayMaya iframe, ensuring that sensitive card information never passes through your application's JavaScript context.

## Parameters

<ParamField path="callback" type="function" required>
  A callback function that will be invoked with the payment token ID when the user submits the credit card form.
  
  **Function Signature:** `(paymentTokenId: string) => void`
  
  **Parameters:**
  - `paymentTokenId` (string): A unique token representing the customer's payment information, generated by PayMaya
</ParamField>

## How It Works

When called, this method:

1. Validates that the SDK has been initialized via `init()`
2. Validates that the callback parameter is a valid function
3. Sets up a `window.addEventListener('message', ...)` listener
4. Listens for postMessage events from the PayMaya iframe
5. Verifies the event origin matches the PayMaya server (sandbox or production)
6. Parses the message data and extracts the `paymentTokenId`
7. Invokes your callback function with the payment token

<Note>
  The method validates the message origin to ensure tokens are only accepted from legitimate PayMaya servers. In sandbox mode, it expects messages from `https://paymayajs-staging.s3.amazonaws.com`. In production mode, it expects messages from `https://paymayajs.s3.amazonaws.com`.
</Note>

## Security Considerations

The transaction handler implements several security measures:

- **Origin Verification**: Only processes messages from the configured PayMaya origin
- **Token-Based Flow**: Receives tokens instead of raw card data
- **Secure iframe Communication**: Uses the postMessage API for cross-origin communication

<Warning>
  **Critical Security Practice:** Always process payments on your server using the payment token. Never attempt to decrypt or store the token on the client side. The token should be immediately sent to your backend via a secure HTTPS connection.
</Warning>

## Basic Usage

```javascript
import PayMaya from 'paymaya-js-sdk';

// Initialize the SDK
PayMaya.init('pk-test-your-public-key', true);

// Create the credit card form
const container = document.getElementById('card-form-container');
PayMaya.createCreditCardForm(container);

// Add transaction handler
PayMaya.addTransactionHandler((paymentTokenId) => {
  console.log('Received payment token:', paymentTokenId);
  
  // Send token to your server
  processPaymentOnServer(paymentTokenId);
});

function processPaymentOnServer(tokenId) {
  fetch('/api/process-payment', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ token: tokenId })
  })
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      window.location.href = '/payment-success';
    } else {
      alert('Payment failed: ' + data.message);
    }
  });
}
```

## Complete Integration Flow

Here's a complete example showing how the credit card form and transaction handler work together:

```javascript
// Step 1: Initialize the SDK
PayMaya.init('pk-test-abc123def456', true);

// Step 2: Create the credit card form
const formContainer = document.getElementById('payment-form');
PayMaya.createCreditCardForm(formContainer, {
  buttonText: 'Pay Now',
  buttonColor: '#007bff',
  buttonTextColor: '#ffffff',
  showLogo: true
});

// Step 3: Add transaction handler to receive token
PayMaya.addTransactionHandler((paymentTokenId) => {
  // Show loading state
  showLoadingIndicator();
  
  // Send token to backend
  fetch('/api/payments/charge', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-CSRF-Token': getCsrfToken()
    },
    body: JSON.stringify({
      paymentToken: paymentTokenId,
      amount: 2500.00,
      currency: 'PHP',
      orderId: 'ORD-12345',
      description: 'Product purchase'
    })
  })
  .then(response => response.json())
  .then(result => {
    hideLoadingIndicator();
    
    if (result.status === 'PAYMENT_SUCCESS') {
      // Redirect to success page
      window.location.href = '/order/success?id=' + result.orderId;
    } else {
      // Show error message
      showErrorMessage('Payment failed: ' + result.message);
    }
  })
  .catch(error => {
    hideLoadingIndicator();
    console.error('Payment error:', error);
    showErrorMessage('An error occurred processing your payment.');
  });
});

function showLoadingIndicator() {
  document.getElementById('loading').style.display = 'block';
}

function hideLoadingIndicator() {
  document.getElementById('loading').style.display = 'none';
}

function showErrorMessage(message) {
  const errorDiv = document.getElementById('error-message');
  errorDiv.textContent = message;
  errorDiv.style.display = 'block';
}

function getCsrfToken() {
  return document.querySelector('meta[name="csrf-token"]').content;
}
```

## Advanced Usage with State Management

```javascript
import PayMaya from 'paymaya-js-sdk';
import { useState } from 'react';

function PaymentForm() {
  const [isProcessing, setIsProcessing] = useState(false);
  const [error, setError] = useState(null);
  
  useEffect(() => {
    // Initialize SDK
    PayMaya.init('pk-test-your-key', true);
    
    // Create form
    const container = document.getElementById('card-form');
    PayMaya.createCreditCardForm(container, {
      buttonText: 'Pay $25.00',
      buttonColor: '#6772e5',
      buttonTextColor: '#ffffff',
      showLogo: true
    });
    
    // Setup transaction handler
    PayMaya.addTransactionHandler(async (paymentTokenId) => {
      setIsProcessing(true);
      setError(null);
      
      try {
        const response = await fetch('/api/charge', {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({
            token: paymentTokenId,
            amount: 2500,
            currency: 'PHP'
          })
        });
        
        const data = await response.json();
        
        if (data.success) {
          // Redirect to success page
          window.location.href = '/success';
        } else {
          setError(data.message || 'Payment failed');
        }
      } catch (err) {
        setError('Network error. Please try again.');
      } finally {
        setIsProcessing(false);
      }
    });
  }, []);
  
  return (
    <div>
      <div id="card-form"></div>
      {isProcessing && <div>Processing payment...</div>}
      {error && <div className="error">{error}</div>}
    </div>
  );
}
```

## Server-Side Integration

After receiving the payment token on your server, use it to charge the customer:

```javascript
// Node.js/Express example
app.post('/api/payments/charge', async (req, res) => {
  const { paymentToken, amount, currency, orderId } = req.body;
  
  try {
    // Use PayMaya's server-side API to charge the token
    const response = await fetch('https://pg-sandbox.paymaya.com/payments/v1/payment-tokens/' + paymentToken + '/payments', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Basic ' + Buffer.from(process.env.PAYMAYA_SECRET_KEY + ':').toString('base64')
      },
      body: JSON.stringify({
        totalAmount: {
          value: amount,
          currency: currency
        },
        requestReferenceNumber: orderId
      })
    });
    
    const paymentResult = await response.json();
    
    if (paymentResult.status === 'PAYMENT_SUCCESS') {
      // Save to database, send confirmation email, etc.
      res.json({
        success: true,
        status: 'PAYMENT_SUCCESS',
        orderId: orderId
      });
    } else {
      res.json({
        success: false,
        message: paymentResult.message || 'Payment failed'
      });
    }
  } catch (error) {
    console.error('Payment processing error:', error);
    res.status(500).json({
      success: false,
      message: 'Server error processing payment'
    });
  }
});
```

## Error Handling

The method includes built-in validation:

- Validates SDK initialization before setting up the listener
- Validates that the callback is a function
- Logs errors to the console if validation fails

```javascript
PayMaya.addTransactionHandler((paymentTokenId) => {
  if (!paymentTokenId) {
    console.error('No payment token received');
    return;
  }
  
  // Process the token
  processPayment(paymentTokenId);
});
```

## Event Flow

1. User fills out credit card form in the iframe
2. User clicks the submit button
3. PayMaya validates and tokenizes the card data
4. PayMaya iframe sends postMessage to parent window
5. `addTransactionHandler` listener receives the message
6. Origin is validated against expected PayMaya domain
7. Payment token is extracted from message data
8. Your callback function is invoked with the token
9. Your code sends the token to your server for processing

## Best Practices

1. **Always validate on the server**: Never trust client-side data alone
2. **Use HTTPS**: Always transmit tokens over secure connections
3. **Handle errors gracefully**: Provide clear feedback to users
4. **Don't store tokens**: Send tokens to your server immediately
5. **Test in sandbox**: Always test the full flow in sandbox mode before going live
6. **Set timeouts**: Consider adding timeout logic for long-running payment processes

<Warning>
  Payment tokens are single-use and time-limited. Do not attempt to reuse a token or store it for later use. Process the payment immediately upon receiving the token.
</Warning>
