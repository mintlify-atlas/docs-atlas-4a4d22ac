  <Steps>
    <Step title="Install the SDK">
      Add the PayMaya JS SDK to your project using npm or yarn:

      <CodeGroup>
      ```bash npm
      npm install paymaya-js-sdk
      ```
      ```bash yarn
      yarn add paymaya-js-sdk
      ```
      ```bash pnpm
      pnpm add paymaya-js-sdk
      ```
      </CodeGroup>

      You can also include it directly via a script tag:
      ```html
      <script src="https://unpkg.com/paymaya-js-sdk@2.0.1/dist/bundle.js"></script>
      ```
    </Step>

    <Step title="Initialize the SDK">
      Import and initialize the SDK with your public API key:

      ```javascript
      import paymaya from 'paymaya-js-sdk';

      // Initialize with your public key
      // Set second parameter to true for sandbox, false for production
      paymaya.init('your-public-key', true);
      ```

      <Note>
        Get your API keys from the [PayMaya Developer Portal](https://developers.paymaya.com). Use sandbox keys for testing.
      </Note>
    </Step>

    <Step title="Create your first payment">
      Use the SDK to create a checkout flow:

      ```javascript
      const checkoutObject = {
        totalAmount: {
          value: 100,
          currency: 'PHP'
        },
        items: [{
          name: 'Test Item',
          quantity: 1,
          totalAmount: { value: 100 }
        }],
        redirectUrl: {
          success: 'https://yoursite.com/success',
          failure: 'https://yoursite.com/failure',
          cancel: 'https://yoursite.com/cancel'
        },
        requestReferenceNumber: '1234567890'
      };

      paymaya.createCheckout(checkoutObject);
      ```

      This will redirect your user to the PayMaya checkout page to complete their payment.
    </Step>
  </Steps>
